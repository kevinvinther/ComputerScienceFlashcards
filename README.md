# Computer Science Flashcards

## How to install
1. Install [anki](https://apps.ankiweb.net/) 
2. Download the add-on [CrowdAnki](https://ankiweb.net/shared/info/1788670778). 
3. Clone the git repository `git clone https://github.com/kevinvinther/ComputerScienceFlashcards`.
4. Download and install [Brain Brew](https://github.com/ohare93/brain-brew/) OR download a release ([releases](https://github.com/kevinvinther/ComputerScienceFlashcards/releases), might be behind)
5. (From here on using BrainBrew) go to the ComputerScienceFlashcards folder
6. Run `brainbrew run recipes/source_to_anki.yaml`
7. Press Files on the Anki toolbar, then CrowdAnki: Import from disk, find `ComputerScienceFlashcards/build` and choose import. 


## How to update
1. Pull the newest update `git pull`
2. Follow steps 4 through 7 in the how to install
